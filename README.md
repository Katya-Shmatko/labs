# Лабораторні роботи 2-6, які реалізують BlockProcessor з SQL-бекендом.
Цей репозиторій містить систему `BlockProcessor` — програму на Python для обробки, валідації та збереження даних про "blocks" та "votes" користувачів. Проект побудований з використанням бази даних SQLite. 
## Структура проекту та етапи розробки:
* Лабораторна робота 2 (`lab2.py`, `lab2.csv`): Читання даних із CSV-файлу. Початкові класи `Block` та `Vote` за допомогою `dataclasses`, додано перевірку шістнадцяткового формату (HEX).
* Лабораторна робота 3 (`lab3.py`): Створено структуру таблиць SQLite (`BLOCKS`, `SOURCES`, `VOTES`, `PERSONS`) та написано скрипт для їх наповнення.
